# PROJECT.md — Room Acoustic Measurement of Northeastern Lecture Halls

## Project Overview

This project measures and compares the acoustic quality of 9 lecture halls and classrooms across Northeastern University's campus. Room impulse responses (IRs) are captured using Exponential Sine Sweeps (ESS) and deconvolved to extract ISO 3382 parameters: T20, T30, D50, C50, D80, C80, and Ts. The goal is to quantify differences in speech intelligibility and reverberant quality across rooms and determine whether those differences are statistically significant.

## Equipment

- **Microphone:** Jounivo JV-601 USB condenser mic (cardioid pattern, uncalibrated). An omnidirectional measurement mic was originally planned but could not be obtained.
- **Speaker:** Portable powered speaker for ESS playback.
- **Laptop:** Same laptop used for all measurements for consistency.
- **Mic stand + clip** for repeatable placement.

### Microphone Limitations

The JV-601 is a cardioid USB mic, not an omnidirectional measurement mic. This has two consequences:

1. **Directional bias:** Reflections arriving from behind the mic are attenuated, which inflates D50/C50 (room appears clearer than an omnidirectional mic would measure). This bias is consistent across all rooms since the same mic was always pointed toward the source.
2. **Uncalibrated frequency response:** The mic has a presence peak around 3–5 kHz typical of vocal condensers. Octave-band analysis is unreliable. Results are reported as broadband or mid-frequency averages (500–2000 Hz band is most trustworthy).

Because the same mic and orientation were used for every measurement, the results are **internally comparable** across rooms even if the absolute values carry bias relative to a calibrated omnidirectional measurement.

## Excitation Signal

A 10-second Exponential Sine Sweep (ESS), 20–20000 Hz, generated at 48 kHz / 24-bit. The sweep file includes 1 s pre-silence and 3 s post-silence (14 s total). Peak level set to −3 dBFS. Generated by `generate_ess.m`.

Parameters: f1 = 20 Hz, f2 = 20000 Hz, R = ln(f2/f1) = 6.9078.

## Measurement Procedure

### Source Placement
- 2 positions per room: front-center (typical lectern) and front-offset
- Height: ~1.5 m (standing speaker height)
- Oriented toward seating area

### Receiver Placement
- 3 positions per source: front, mid, and back of seating
- Height: ~1.2 m (seated ear height)
- At least 1 m from walls, at least 2 m from source
- Mic pointed toward source (consistent across all positions)

### Recording Protocol
- Record ESS playback + room response as a single wav file
- Room empty, doors closed, HVAC state documented
- No clipping verified before proceeding
- Quality check: direct sound visible, decay tail present

### Deconvolution Method

The Farina analytical inverse filter is used. For an ESS with sweep rate R = ln(f2/f1):

1. Strip silence padding from the input signal (5% peak threshold)
2. Time-reverse the active sweep
3. Multiply by exponential decay envelope: exp(−n·R/(N−1))
4. Normalize so auto-deconvolution (sweep ⊛ inverse) peaks at 1
5. Convolve recorded output with inverse filter via FFT (zero-padded to avoid circular convolution)
6. Locate direct-sound peak, extract IR window

This method avoids spectral division and has no regularization parameters to tune. It also separates harmonic distortion products from the linear IR (they appear at negative times and are windowed out).

f1 and f2 are specified explicitly in the room metadata (preferred) or auto-estimated from zero-crossing periods in the time domain.

### Lundeby Truncation

The Lundeby algorithm (1995) iteratively estimates the noise floor and finds the truncation point where the IR decay meets it. The averaging window is set adaptively: a coarse T60 estimate determines the window length (T60/30, clamped to 2–50 ms). This adapts automatically to any room without manual tuning.

After truncation, Schroeder backward integration with the Chu (1978) noise-energy correction produces the Energy Decay Curve (EDC).

### Room Acoustic Parameters

All per ISO 3382:

- **T20, T30:** Reverberation time from linear regression on EDC (−5 to −25 dB and −5 to −35 dB, extrapolated to −60 dB)
- **D50:** Definition — fraction of energy in first 50 ms (speech intelligibility)
- **C50:** Clarity at 50 ms — early/late energy ratio in dB
- **D80, C80:** Same at 80 ms boundary (music clarity)
- **Ts:** Centre time — energy-weighted mean arrival time

## Statistical Analysis Plan

For each room: mean, standard deviation, and 95% confidence interval of each parameter across the 6 source-receiver combinations.

Across rooms: one-way ANOVA for each parameter to test whether room differences are statistically significant. Null hypothesis: all room means are equal. Alternative: at least one room differs.

Expected ranges for lecture halls (from ISO 3382-1 and literature):

| Parameter | Low | High |
|---|---|---|
| T20/T30 | 0.4 s | 1.2 s |
| D50 | 0.3 | 0.7 |
| C50 | −5 dB | 5 dB |
| C80 | −5 dB | 5 dB |
| Ts | 60 ms | 260 ms |

## Validation

The analysis pipeline was validated against three reference datasets before applying to campus data:

1. **Zenodo ESS dataset** (Stolz et al., TU Ilmenau, DOI: 10.5281/zenodo.15010963): 8 s and 64 s sweeps in a 0.25 s T60 room. Both sweep lengths produced consistent T20 (0.211 s), D50 (97.9%), confirming the deconvolution and parameter extraction are correct.

2. **Pyramic dataset** (EPFL, DOI: 10.5281/zenodo.1209563): Anechoic chamber recording. Confirmed near-zero reverberation (D50 ≈ 100%, Ts ≈ 1.7 ms).

3. **Synthetic ground truth**: Known RIR convolved with generated ESS + additive noise. Quantitative comparison via normalised misalignment (NM) and cross-correlation. Confirmed pipeline accuracy within the Farina method's inherent ~−11 dB sidelobe floor.

## Known Limitations

- Cardioid mic biases D50/C50 high relative to omnidirectional measurement (see Equipment section)
- Low-frequency response of both mic and portable speaker is limited; octave-band analysis below 250 Hz is unreliable
- Single source type (portable speaker) rather than dodecahedron omnidirectional source per ISO 3382-1
- No acoustic calibrator available; absolute SPL levels not calibrated
- Farina inverse filter has ~−16 dB sidelobe floor from discrete-time approximation; does not affect broadband parameters for rooms with T60 > 0.3 s
